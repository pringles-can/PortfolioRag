# Interview Answer — Cloud Experience

## Summary

Steven has used AWS and Terraform to build cloud infrastructure for healthcare data workflows.

## AWS Experience

- AWS API Gateway
- AWS Lambda
- AWS S3
- AWS EC2
- AWS ECS
- AWS SAM

## Terraform Experience

Steven used Terraform to create AWS infrastructure, Kafka topics, Kafka producer groups, Kafka consumer groups, API Gateway resources, Lambda resources, and S3 buckets.

## Example Answer

I've used AWS mainly in backend and event-driven healthcare systems. At CareMetX, I stood up AWS API Gateway, Lambda, S3 buckets, and Kafka infrastructure using Terraform. One project involved consuming legacy SOAP requests, publishing them to Kafka, mapping data to FHIR responses, and publishing those responses to downstream Kafka topics. Another project used Lambda triggered by status changes to transform prescription and enrollment data into Benefit Verifications and Prior Authorizations.
