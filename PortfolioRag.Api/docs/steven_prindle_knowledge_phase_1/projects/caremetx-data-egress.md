# CareMetX — Data Egress & Transformation

## Role

Senior Software Engineer

## Company

Daugherty Business Solutions / CGI

## Dates

June 2023 – April 2024

## Overview

Steven helped establish a foundational cloud transformation pattern for converting incoming healthcare data into outgoing Benefit Verifications and Prior Authorizations.

## Business and Technical Significance

This project represented the organization's first cloud venture for transforming incoming data into outgoing healthcare verification and authorization workflows.

## Responsibilities

- Established foundational cloud transformation patterns.
- Implemented a state machine using Stateless in C#.
- Managed prescription and enrollment status changes.
- Transformed data into Benefit Verifications and Prior Authorizations.
- Used AutoMapper for object mapping.
- Stood up AWS Lambda.
- Created Kafka topics and producer groups with Terraform.
- Triggered Lambda functions from status changes.
- Mapped data to FHIR using JSONata.
- Published transformed data to Kafka topics.

## Performance Improvement

Steven identified a solution to a work-stopping timeout and helped produce 80% faster load times on common pages.

## Data and Persistence Work

- Refactored legacy Hibernate expressions to use lazy loading.
- Created an Oracle view to manage complex data sets derived from multiple database transactions.

## Mentoring

Steven mentored a junior developer in Git using Bash and LINQ-to-SQL best practices.
