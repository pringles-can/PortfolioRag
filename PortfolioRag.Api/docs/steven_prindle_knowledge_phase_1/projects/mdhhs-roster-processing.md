# MDHHS — Roster Processing and Batch Billing

## Role

Full Stack Software Developer

## Company

Kunz, Leigh & Associates

## Dates

September 2017 – October 2019

## Client

Michigan Department of Health and Human Services

## Overview

Steven supported custom roster processing and batch billing workflows for MDHHS.

## Responsibilities

- Coordinated directly with payors to build custom file formats when EDI was not an option.
- Processed incoming roster files using Bash scripts.
- Supported custom header formats developed with payors.
- Used Bash scripts to read file headers and execute specific Groovy scripts.
- Migrated legacy Oracle batch billing jobs to Groovy scripts.

## Business Impact

Steven helped reduce legacy Oracle batch billing runtime from 10–12 hours to approximately 30 minutes.
