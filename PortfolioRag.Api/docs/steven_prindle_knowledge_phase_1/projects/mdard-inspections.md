# MDARD — Agriculture Inspections Platform

## Role

Full Stack Software Developer

## Company

Kunz, Leigh & Associates

## Dates

October 2019 – January 2023

## Client

Michigan Department of Agriculture & Rural Development

## Overview

Steven worked on an inspections platform for MDARD, supporting plant and pesticide inspections.

## Responsibilities

- Created Angular providers and component injectors.
- Set a foundation for dependency injection.
- Simplified unit testing.
- Built a Storybook application for rapid UI prototyping by Business Analysts.
- Integrated a proprietary Angular Material library.
- Developed base UI templates.
- Built a custom service worker to enable local offline caching.
- Used Angular Material for accessibility and responsive design.

## Business Impact

The system doubled the number of inspections completed by state inspectors from approximately 1,000 to more than 2,000.
